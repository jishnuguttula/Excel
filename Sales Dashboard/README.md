# 📊 Excel Sales Dashboard

## 📌 Project Overview

This project is an interactive **Sales Dashboard** built using Microsoft Excel.

It analyzes sales data and provides insights into:

* Total Sales
* Total Profit
* Sales by Region
* Sales by Category
* Sales Trends over time

---


## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Charts (Bar, Pie, Line)
* Slicers
* KPI Cards

---

## 📊 Key Features

* Interactive filters using slicers
* Dynamic KPI cards
* Trend analysis using line charts
* Category and region-wise breakdown

---

## 📈 Insights

* Tech category generates the highest revenue
* East region shows strong performance
* Laptop is the most profitable product

---

## 🚀 How to Use

1. Open the Excel file
2. Use slicers to filter data
3. Analyze charts and KPIs
